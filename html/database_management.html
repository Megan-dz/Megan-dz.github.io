Database Management with Python

Python makes it easy to manage databases using libraries like sqlite3 for lightweight databases or SQLAlchemy for more complex ones.

Using sqlite3

SQLite is a built-in module for file-based databases. Example to create and insert data:

import sqlite3

conn = sqlite3.connect('example.db')
c = conn.cursor()
c.execute('''CREATE TABLE IF NOT EXISTS users (id INTEGER PRIMARY KEY, name TEXT)''')
c.execute("INSERT INTO users (name) VALUES ('Alice')")
conn.commit()
conn.close()

Querying Data

To fetch data: c.execute("SELECT * FROM users") and then rows = c.fetchall().

Advanced Libraries

For larger databases like PostgreSQL or MySQL, use libraries like psycopg2 or pymysql, or ORM like SQLAlchemy for object-relational mapping.
