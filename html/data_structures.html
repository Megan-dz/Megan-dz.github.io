Core Data Structures in Python

Python provides several built-in data structures for storing and manipulating data efficiently.

Lists

Lists are mutable sequences. Example: my_list = [1, 2, 3]. You can add, remove, or modify elements.

Tuples

Tuples are immutable sequences. Example: my_tuple = (1, 2, 3). Useful for fixed data.

Dictionaries

Dictionaries store key-value pairs. Example: my_dict = {'name': 'Alice', 'age': 30}. Fast lookups by key.

Sets

Sets are unordered collections of unique items. Example: my_set = {1, 2, 3}. Great for membership testing and removing duplicates.
