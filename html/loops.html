Loops - For / While Nested Loops

In Python, loops are used to execute a block of code repeatedly. There are two main types: for loops and while loops.

For Loops

A for loop iterates over a sequence (like a list, tuple, or range). Example:

for i in range(5):
    print(i)

This prints numbers 0 to 4.

While Loops

A while loop runs as long as a condition is true. Example:

count = 0
while count < 5:
    print(count)
    count += 1

This also prints numbers 0 to 4.

Nested Loops

Nested loops are loops inside other loops, useful for multi-dimensional data. Example:

for i in range(3):
    for j in range(3):
        print(i, j)

This prints pairs like (0,0) to (2,2).
