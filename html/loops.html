Python Loops – For, While & Nested Loops
Study Notes

Loops allow us to repeat a block of code multiple times — essential for processing lists, generating patterns, reading data, games, simulations, and more.

1. For Loops

The for loop is used to iterate over a sequence (list, tuple, string, range, dictionary keys, etc.).

Basic Syntax

for variable in sequence:
    # code block (indented)
    print(variable)

Most Common Use: range()

# 0 to 4
for i in range(5):
    print(i)

# 2 to 9
for i in range(2, 10):
    print(i)

# Even numbers 0–10
for i in range(0, 11, 2):
    print(i)    # 0 2 4 6 8 10

Iterating over different types

# List
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# String
for char in "Python":
    print(char)     # P y t h o n

# Dictionary (keys by default)
person = {"name": "Megan", "city": "Bengaluru"}
for key in person:
    print(key, "→", person[key])
Tip: Use for item in collection when you want to work directly with each element.
Use for i in range(len(collection)) when you need the index.

2. While Loops

The while loop runs as long as a condition is True. Great when you don’t know in advance how many times you need to loop.

Classic Examples

# Countdown
count = 5
while count > 0:
    print(count)
    count -= 1
print("Blast off!")
# Keep asking until correct
password = ""
while password != "python123":
    password = input("Enter password: ")
print("Access granted!")
Warning – Infinite Loop Danger!
Always make sure the condition will eventually become False.
Common mistake: 
x = 5
while x > 0:
    print(x)    # Forgot to decrease x → infinite loop!

3. Loop Control Statements

Statement What it does Example
break Exit the loop immediately Stop when found: for num in numbers: if num == target: break
continue Skip rest of current iteration, go to next Skip even: if i % 2 == 0: continue
pass Do nothing (placeholder) if condition: pass
else clause Runs if loop finished without break Search: “not found” message

Example with else

numbers = [4, 7, 2, 9, 1]
target = 10

for num in numbers:
    if num == target:
        print("Found!")
        break
else:
    print("Target not found in list")

4. Nested Loops

A loop inside another loop — very common for working with 2D data (matrices, tables, grids, patterns).

Multiplication Table Example

for i in range(1, 6):
    for j in range(1, 6):
        print(f"{i*j:4}", end=" ")
    print()

Star Pattern Example

for i in range(5):
    for j in range(i + 1):
        print("*", end=" ")
    print()
